# 🤖 AI Chatbot - اردو میں سیٹ اپ گائیڈ

## 📋 ڈیٹا بیس کی تفصیلات

**موجودہ ڈیٹا بیس کی معلومات:**
- ڈیٹا بیس کا نام: `rsk0_08`
- یوزر نام: `rsk0_08`
- پاس ورڈ: `123456`
- ہوسٹ: `localhost`

---

## 🚀 انسٹالیشن کے قدم

### 1️⃣ ڈیٹا بیس امپورٹ کریں
1. phpMyAdmin کھولیں
2. `rsk0_08` ڈیٹا بیس بنائیں (اگر نہیں ہے)
3. `database.sql` فائل امپورٹ کریں
4. `messages` ٹیبل خودکار بن جائے گی

### 2️⃣ AI API سیٹ اپ کریں (مفت!)

آپ کا چیٹ بوٹ اب **3 مفت AI APIs** سپورٹ کرتا ہے۔ کوئی ایک منتخب کریں:

---

#### ⚡ **آپشن 1: Groq API (تجویز کردہ - سب سے تیز)**

**Groq کیوں؟**
- ✅ مکمل طور پر مفت
- ✅ بہت تیز جوابات
- ✅ GPT-4 جیسی ذہانت
- ✅ روزانہ 14,400 درخواستیں مفت
- ✅ کریڈٹ کارڈ کی ضرورت نہیں

**سیٹ اپ:**
1. یہاں جائیں: https://console.groq.com
2. مفت اکاؤنٹ بنائیں
3. API Keys سیکشن میں جائیں
4. نئی API key بنائیں (شروع ہوتی ہے `gsk_` سے)
5. `api/chat_handler.php` فائل کھولیں
6. یہ لائن تلاش کریں: `$groqApiKey = 'gsk_YOUR_GROQ_API_KEY_HERE';`
7. اپنی اصل key سے بدلیں

---

#### 🌐 **آپشن 2: Cohere API**

**سیٹ اپ:**
1. یہاں جائیں: https://cohere.com
2. مفت سائن اپ کریں
3. API Keys کاپی کریں
4. `api/chat_handler.php` میں `$cohereApiKey` کی جگہ لگائیں

---

#### 🤗 **آپشن 3: Hugging Face**

**سیٹ اپ:**
1. یہاں جائیں: https://huggingface.co/settings/tokens
2. مفت اکاؤنٹ بنائیں
3. نیا ٹوکن بنائیں (شروع ہوتا ہے `hf_` سے)
4. `api/chat_handler.php` میں `$hfApiKey` کی جگہ لگائیں

---

### 3️⃣ اپنے چیٹ بوٹ کو ٹیسٹ کریں

1. تمام فائلیں اپنے ویب سرور پر اپ لوڈ کریں
2. `index.php` براؤزر میں کھولیں
3. بات چیت شروع کریں!

## 🔧 مسائل کا حل

### چیٹ بوٹ جواب نہیں دے رہا؟
1. `db.php` میں ڈیٹا بیس کنکشن چیک کریں
2. ڈیٹا بیس کی اسناد کی تصدیق کریں
3. یقینی بنائیں کہ `messages` ٹیبل موجود ہے

### بہتر جوابات چاہتے ہیں؟
1. مفت API keys میں سے ایک سیٹ اپ کریں (Groq تجویز کردہ)
2. فال بیک علمی بنیاد کام کرتی ہے، لیکن APIs بہت بہتر جوابات فراہم کرتے ہیں

### ڈیٹا بیس کی خرابیاں؟
1. `database.sql` فائل امپورٹ کریں
2. چیک کریں کہ ڈیٹا بیس کا نام `rsk0_08` ہے
3. تصدیق کریں کہ صارف کو مناسب اجازتیں ہیں
